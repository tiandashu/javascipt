把JavaScript插入到HTML页面需要使用scrip元素。使用这个元素可以把JavaScript嵌入到HTML页面中，让脚本和标记混合在一起；也可以混合外部的JavaScript文件。需要注意的是：

  • 在包含外部的JavaScript文件时，src必须指向对应的文件url。而这个文件可以是与包含它的页面位于同一个服务器，也可以是其他任何一个域中的文件。
  • 所有 script 都会按照他们在页面中出现的先后顺序依次被解析。在不适用defer和async的情况下，只有在解析完前面的 script 元素中的代码之后，才会开始解析后面 script 元素中的代码
  • 由于浏览器会先解析完不使用defer属性的 script 中的代码，然后再解析后面的内容，所以一般应该把 script 元素放在页面最后，即主要内容的后面，/body 标签的前面
  • 使用defer属性可以让脚本在文档完全呈现之后再执行。延迟脚本总是按照指定它们的顺序执行。
  • 使用async属性表示当前脚本不必等待其它脚本，也不必阻塞文档的呈现。但是不能保证异步脚本按照它们在页面中出现的位置顺序执行。
  • async和defer相同的是都只对外部脚本生效，不同的是设置async不能保证按照指定脚本的先后顺序执行。defer按照脚本的先后顺序依次执行。
  • 使用noscript元素包裹的内容，只有在浏览器不支持JavaScript或者禁用JavaScript的情况下才显示
